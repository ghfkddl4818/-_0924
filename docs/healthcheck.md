# Healthcheck 지침

## 실행 전 점검
- `templates/` 경로에 GUI 매칭용 이미지 템플릿이 모두 배치되어 있는지 확인합니다.
- `TESSERACT_CMD` 환경 변수가 시스템의 tesseract 실행 파일을 가리키는지 확인합니다.
- `GOOGLE_APPLICATION_CREDENTIALS` 환경 변수가 Vertex 호출에 사용할 서비스 계정 키를 참조하는지 확인합니다.
- `config/config.yaml`의 모델 이름, Vertex 프로젝트 ID, 리전 값이 운영 환경과 일치하는지 검토합니다.

## GUI 헬스체크 실패 시 대응
- 템플릿 이미지가 누락되었거나 이름이 변경되지 않았는지 확인하고 필요 시 재배포합니다.
- 서비스 계정이나 로컬 사용자 계정에 필요한 파일/폴더 접근 권한이 있는지 검증합니다.
- 실행 환경 해상도가 기준보다 낮거나 DPI 설정이 맞지 않은 경우 조정 후 재시도합니다.
